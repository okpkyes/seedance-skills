# Seedance Skills

专为字节跳动**即梦平台 Seedance 2.0**设计的 AI 视频提示词工程 Skill，适用于 [WorkBuddy](https://www.codebuddy.cn/) 等 AI 助手平台。

> 从一句模糊需求到可直接粘贴使用的电影级视频提示词。

---

## 核心理念

**"写意图、不写细节"** —— Seedance 2.0 具备世界知识和导演思维，在有世界知识的场景下只需描述意图，让 AI 发挥创造力。仅在需要精确控制分镜、角色一致性或特殊运镜时才展开详细描述。

---

## 功能总览

### 1. 提示词构思三法

解决「需求 → 意图清晰」的转化问题：

| 方法 | 适用场景 | 核心思路 |
|------|---------|---------|
| **自问式** | 心里有画面但说不清 | 让 AI 先反问拆解需求，再产出精准提示词 |
| **拆分法** | 一次生成无法满足多元素 | 大问题拆成多个小问题分别生成再拼合 |
| **精确描述法** | 构图/视角需要精确控制 | 只写画面中实际存在的东西 + 精确位置 |

### 2. 结构化控片公式（7 维指令链）

解决「意图清晰 → 视频不崩」的问题：

```
【精准主体】+【动作细节】+【场景环境】+【光影色调】+【镜头运镜】+【视觉风格】+【约束条件】
```

搭配 3 大避坑策略：
- **数字锁角色锚定** — 用 `<主体1>` 替代冗余容貌描写，锁定角色一致性
- **时间轴分镜解构** — 将叙事拆为精确时间轴分镜表，逐帧控制
- **微动态物理降维** — 大动态拆为连续微动态，规避穿模和骨骼错位

### 4. TVC 广告分镜系统

品牌 TVC 广告专用，**5 镜头 × 3 秒**分镜脚本设计：

| 镜头 | 名称 | 目的 |
|------|------|------|
| SC1 | 状态引入 | 呈现痛点/环境/初始状态 |
| SC2 | 核心交互 | 产品介入，展示核心物理细节 |
| SC3 | 视觉奇观 | 极致光影/材质转换/微观特写 |
| SC4 | 情绪释放 | 人物情绪反馈或环境终极质感 |
| SC5 | 品牌落版 | 产品展示 + Slogan/Logo |

内置饮料/美妆/电子/食品四大品类的场景适配规则和镜头语言组合。

### 6. AI 短剧全流程工作流（文-资-视-剪）

将小说/故事转为多集视频的完整管线：

| 阶段 | 核心 | 作用 |
|------|------|------|
| **文** | 剧本创作 + 分镜拆解 | 故事蓝图与施工图纸 |
| **资** | 角色/场景/道具资产库 | 保证成片一致性 |
| **视** | 多模态直出视频 | 角色+场景+剧本驱动生成 |
| **剪** | 后期剪辑与画面修补 | 最终视听整合 |

包含素材编号规范（C/S/P 前缀）、分镜剧本格式、资产生成规则、集数衔接策略。

### 7. 超长视频分段拼接

> 15 秒以上视频的分段生成 + 视频延长拼接方案。

- 自动按叙事节奏切分，每段 ≤ 15 秒
- 段间画面衔接点设计
- 序列续写纪律（观测状态覆盖计划状态、被拒片段不入正典等）

### 8. 2 秒钩子框架

10 种留存钩子模式，专治前 2-3 秒完播率：
强反差 / 强冲突 / 强悬念 / 强情绪 / 视觉奇观 / 身份反差 / 禁忌破规 / 数字承诺 / 第一人称代入 / 声音钩子。

## @引用系统

| 素材类型 | 引用格式 | 数量上限 |
|---------|---------|---------|
| 图片 | `@图片1` ~ `@图片9` | ≤ 9 张 |
| 视频 | `@视频1` ~ `@视频3` | ≤ 3 个 |
| 音频 | `@音频1` ~ `@音频3` | ≤ 3 个 |

支持 14 种引用用途标注（首帧/尾帧/人物形象/场景/运镜/动作/特效/节奏/音色/BGM/音效/服装/产品/字体），多引用可自由组合。

## 高级技巧

- **时间戳分镜法** — `0-3秒：... 4-8秒：...` 精确控制 15 秒长视频
- **四段式公式** — `【风格】+【时间轴】+【声音】+【参考】` 标准化输出
- **导演一致性法则** — 每段提示词锁定一个核心意图，运镜/灯光/表演/声音全部服务它
- **序列续写纪律** — 续写前回顾上一段真实尾帧，被拒片段不入正典
- **版权规避** — 用 "Figure 1" 替代 IP 角色名，描述外观特征而非角色名

## 平台参数速查

| 维度 | 规格 |
|------|------|
| 图片输入 | jpeg/png/webp/bmp/tiff/gif，≤ 9 张，单张 < 30MB |
| 视频输入 | mp4/mov，≤ 3 个，总时长 2-15 秒，单个 < 50MB |
| 音频输入 | mp3/wav，≤ 3 个，总时长 ≤ 15 秒，单个 < 15MB |
| 混合上限 | 最多 12 个文件（图片+视频+音频合计） |
| 生成时长 | 4-15 秒，可自由选择 |
| 分辨率 | 支持 2K 输出 |
| 声音输出 | 自带音效/配乐 |

## 使用方式

### 在 WorkBuddy 中使用

将整个目录复制到 `~/.workbuddy/skills/seedance/`，重启 WorkBuddy 即可自动加载。

触发词：`生成视频提示词` / `Seedance` / `即梦` / `视频提示词` / `AI视频` / `短剧` / `广告视频` 等。

### 在其他平台使用

直接阅读 `SKILL.md` 作为系统提示词，按需查阅 `references/` 下的参考文件即可。
