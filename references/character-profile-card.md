# 人物角色卡（韩系偶像风·character profile 极简杂志排版）

一张图把角色所有要素塞进去：基础信息 + 三视图 + 6 表情 + 服装拆解 + 配饰 + 细节特写 + 色卡 + 人物文案。

## 完整提示词

```
竖版韩系偶像角色设定卡，character profile，极简杂志风排版，纯白无影背景，

高级轻奢版式，细分割线，信息模块化分区，

顶部都是超大衬线字体英文姓名标题，搭配中英文文艺标语，

右上角标注编号 No.07，


上半分部三栏布局：

- 左栏：基础信息区，标注姓名、职业、年龄、身高、生日、星座、外貌特征、气质关键词与星级评分
- 中栏：人物三视图，同一人物外貌描述："的年轻女生，身穿【具体服装描述】"，标准站姿，分别展示正面、正侧面、背面全身立绘，韩系爱豆颜值，精致五官，皮肤通透，身材纤长比例优越
- 右栏：6 格表情头像，同一人物的大头肖像，分别呈现平静、微笑、眨眼、认真、惊讶、思考六种表情，妆容统一


下半分部四栏布局：

- 第一栏：服装拆解，铺展示整套穿搭的所有单品，白底整齐排列
- 第二栏：配饰展示，陈列耳饰、项链、发饰等搭配小件
- 第三栏：细节特写，4 张小图分别展示眼睛、嘴唇、发丝、服装面料细节
- 第四栏：配色色卡，垂直排列的色块附带十六进制色号，提取整套造型的主题配色


底部三栏：

- 中英文人物介绍文案
- 圆角矩形关键词标签
- 手写体英文签名
- 附小字标语


无影棚光，均匀打光，超写实摄影质感，高清细节，8k，极致清晰，

整体色调统一为【主色调】色系，干净高级的视觉效果。
```

## 信息架构速查

| 区域 | 内容 | 排版要点 |
|---|---|---|
| 顶部 | 大衬线英文姓名 + 中英文文艺标语 + 编号 No.07 | 标题大、留白多 |
| 上左 | 基础信息（姓名/职业/年龄/身高/生日/星座/外貌/气质/星级） | 关键词列表式 |
| 上中 | 三视图（正面/正侧面/背面） | 全身立绘 + 服装描述 |
| 上右 | 6 格表情头像（平静/微笑/眨眼/认真/惊讶/思考） | 统一妆容 |
| 下1 | 服装拆解（整套穿搭单品） | 白底平铺 |
| 下2 | 配饰（耳饰/项链/发饰） | 陈列式 |
| 下3 | 4 张细节特写（眼/唇/发丝/面料） | 小图 |
| 下4 | 配色色卡（垂直排列 + 十六进制色号） | 提取主配色 |
| 底部 | 中英文介绍 + 关键词标签 + 手写签名 | 杂志风收尾 |

## 拆解要点

| 维度 | 固定写法 |
|---|---|
| 画幅 | 竖版 9:16（适合手机屏幕浏览） |
| 背景 | 纯白无影背景，均匀打光 |
| 风格 | 韩系偶像 + 极简杂志 + 高级轻奢 |
| 信息密度 | 高（一次集成 8 类角色要素） |
| 字体 | 顶部大衬线英文 + 关键词小标签 + 手写体签名 |
| 视觉统一 | 整体色调统一为【主色调】（变量） |
| 质感 | 超写实摄影 + 8k + 极致清晰 |

## 使用方式

1. 替换变量：`{{姓名}}` `{{职业}}` `{{年龄}}` `{{身高}}` `{{生日}}` `{{星座}}` `{{外貌特征}}` `{{气质关键词}}` `{{服装描述}}` `{{主色调}}` `{{文艺标语}}` `{{中英文介绍}}`。
2. 适用于：
   - AI 漫剧/短剧角色定档
   - OC（原创角色）资料卡
   - 模卡/演员定妆照
   - 作品集"角色世界"页
   - 同人/二创角色介绍图
3. 配套输出建议：先做 → `face-three-view.md` 锁定形象 → 再做本卡。
