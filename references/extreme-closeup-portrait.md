# 极致半脸特写·单眼视觉中心（超写实微距人像）

> 用途：极近距离电影级半脸特写，只露出面部一侧，以**一只眼睛为绝对视觉中心**。
> 核心：微距超写实 + RAW 未磨皮质感 + 真实瑕疵保留，规避 AI 塑料感。
> 触发词：半脸特写、单眼特写、极致特写、微距人像、eye close-up、超写实半脸、documentary portrait、面部瑕疵可见。

---

## 一、模板定位

这是一个**成品级构图模板**，与方法论类 reference 配合使用：
- 结构自检 → `portrait-prompt-philosophy.md`（5 层心智模型）
- 皮肤质感词 → `skin-texture-banks.md`
- 光影结构词 → `portrait-lighting-banks.md`
- 去塑料感诊断 → `anti-plastic-realism.md`
- 镜头/调色词 → `cinematic-vocabulary.md`

区别于其它人像模板：本模板锁死「极近距离 + 半张脸 + 单眼为焦点中心」这一独特构图，追求编辑摄影 / 博物馆级写实。

## 二、合规前置（强制）

- 主体**必须为成年人**。若用户用占位符（如"x岁""某性别"），一律先澄清或按**成年人**处理，禁止生成未成年人写实人脸。
- 写实真人脸：本模板是**文生图文字描述**，不涉及上传真人素材，不受平台"禁止上传真人脸"限制。

## 三、成品 Prompt（占位符可替换）

> 一位{{年龄段}}的{{性别}}，极近距离半张脸特写肖像，只露出面部一侧，一只眼睛作为绝对视觉中心直视镜头。超写实人像摄影，纪录片质感，RAW格式原始风味。自然真实的皮肤纹理：可见细腻毛孔、细软汗毛、淡淡雀斑与不均匀肤色，面部微小瑕疵与不完美肌理真实保留；虹膜纤维清晰分明，眼球温润带有真实环境反光，睫毛根根分明，浓密眉毛略带杂毛，唇部纹理细致可见。{{光影}}，勾勒清晰面部骨骼结构，轻微轮廓光勾边，自然阴影过渡柔和。极浅景深，背景柔和虚化。微距人像摄影，85mm微距镜头、f/1.2大光圈、全画幅相机，眼部焦点极其锐利，细腻微对比，高级电影调色，真实色彩科学，柔和胶片颗粒，未经磨皮与任何美颜滤镜处理，自然解剖结构准确，真实发丝与{{面部毛发}}细节，克制低饱和色彩，高宽容度，博物馆级写实人像、获奖级编辑摄影质感。
>
> 禁止：磨皮、美颜滤镜、光滑无瑕皮肤、AI塑料感、过度锐化、字幕/文字/水印/Logo。

## 四、占位符词库

| 占位符 | 默认值 | 候选词 |

## 五、技术规格内核（固定不建议改）

- 镜头：85mm 微距，f/1.2 大光圈，全画幅
- 焦点：眼部极其锐利，其余极浅景深虚化
- 质感：RAW / 柔和胶片颗粒 / 细腻微对比 / 高宽容度 / 克制低饱和
- 反向约束：未磨皮、无美颜滤镜、无塑料感、无字幕水印

## 六、参数建议

- 尺寸：竖版 1024×1536（人脸特写最佳）；亦可 16:9 横版做电影感。
- 质量：high。
- 迭代：效果偏"假滑"时，仅加强 §五 反向约束与 `skin-texture-banks.md` 的必加词，遵循「一次只改一个变量」。

## 七、验证记录

- 2026-07-20：20多岁成年男性半脸特写，1024×1536 / high，质感达标（毛孔、胡茬、虹膜纤维清晰，无塑料感）。
