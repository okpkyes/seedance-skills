# 概念先行 Prompting

避免"技术词堆砌但空洞"问题：先定概念，再写提示词。

---

## 问题诊断

**症状**：
- 提示词写了很多技术词（"电影感""4K""浅景深""环绕拍摄"）
- 但生成的视频"没有主题""不知道在拍什么""画面很空"
- 视觉效果还行，但没有叙事、没有情绪、没有灵魂

**根本原因**：
- 只描述了"怎么拍"（技术层面）
- 没有描述"为什么要拍这个"（概念层面）
- AI 生成了技术正确的画面，但没有灵魂

---

## 解决方案：概念先行 3 步法

### Step 1：定义一句话 Concept

**在写提示词之前，先回答这个问题：**

```
这个视频的核心概念是什么？
用一句话说清楚。
```

**示例**：

| ❌ 错误（没有概念） | ✅ 正确（有概念） |
|-------------------|-----------------|
| "一个很酷的打斗场景" | "一个被背叛的弟子，在山顶与师父决裂" |
| "美女在跳舞" | "一个女特工在谍战中执行任务，表面冷静内心紧张" |
| "产品展示" | "一个可乐广告，强调夏日清凉感和畅快体验" |

**概念要素**：
- **谁**（角色/产品）
- **在哪里**（场景）
- **做什么**（动作/情节）
- **为什么**（动机/情绪/主题）

---

### Step 2：设计 3-Beat 叙事弧

**即使是 15 秒视频，也要有叙事弧。**

**3-Beat 结构**：
```
Beat 1（开头 0-3秒）：铺垫 / 建立情绪 / 展示现状
Beat 2（中段 4-10秒）：冲突 / 动作 / 变化
Beat 3（结尾 11-15秒）：高潮 / 反转 / 定格情绪
```

**示例 - 霸总短剧**：
```
Concept："女主撕契约反杀，从卑微到强势"

Beat 1（0-3秒）：女主撕契约，总裁阻拦（铺垫：权力关系）
Beat 2（4-10秒）：女主扔契约纸，台词反杀（冲突：权力反转）
Beat 3（11-15秒）：女主转身离开，总裁僵住（高潮：反杀完成）
```

**示例 - 产品广告**：
```
Concept："可乐的夏日清凉感，从视觉到听觉的畅快体验"

Beat 1（0-3秒）：热天，角色口渴，看到可乐（铺垫：需求）
Beat 2（4-10秒）：开瓶，倒可乐，冰块碰撞（动作：满足需求）
Beat 3（11-15秒）：喝下可乐，满足表情，气泡特效（高潮：畅快体验）
```

---

### Step 3：为每个 Beat 做"存在性测试"

**在写具体提示词之前，问自己：**

```
这个镜头为什么要存在？
它服务于 Concept 的哪个部分？
如果删掉这个镜头，Concept 还完整吗？
```

**存在性测试示例**：

| 镜头描述 | 存在性测试 | 结果 |
|---------|-----------|------|
| "低角度特写主角衣摆" | 服务于 Concept 的"史诗感"？是 | ✅ 保留 |
| "镜头环绕360度" | 服务于叙事？还是炫技？炫技 | ❌ 删除或简化 |
| "背景有霓虹灯光" | 服务于"赛博朋克"概念？是 | ✅ 保留 |
| "浅景深，背景虚化" | 突出主体？是 | ✅ 保留，但别只写这个 |

**原则**：
- 每个镜头/技术词，都要服务于 Concept
- 如果某个技术词不服务于 Concept，删掉
- 技术词是"怎么拍"，Concept 是"为什么要拍"

## 完整示例：从 Concept 到提示词

### 示例 1：仙侠战斗（概念先行版）

**Step 1：定义 Concept**
```
Concept："一个被背叛的剑修，在魔气入侵的仙山上，独自迎敌，
         从孤独到坚定的情绪转变"
```

**Step 2：设计 3-Beat 叙事弧**
```
Beat 1（0-3秒）：仙山被魔气入侵，剑修独自站在山巅（孤独）
Beat 2（4-10秒）：魔兽来袭，剑修拔剑迎敌（冲突）
Beat 3（11-15秒）：剑修斩退魔兽，眼神坚定（坚定）
```

**Step 3：存在性测试 + 写提示词**
```
15秒仙侠镜头，金红暖色调，

0-3秒：俯拍云海翻涌中的仙山全景，魔气升腾（服务于 Beat 1：仙山被入侵）
镜头缓缓下推穿过云层，剑修背对镜头站在悬崖边（孤独感），
衣袍被风吹动（存在性测试：✅ 展示环境+角色状态），

4-8秒：剑修缓缓转身面向镜头，拔剑出鞘（Beat 2：冲突开始），
剑刃金光闪烁，目光坚毅（存在性测试：✅ 情绪转变），
低声道"今日，便以这柄剑，镇尔等邪祟！"（台词服务于 Concept：坚定），

9-15秒：魔兽从魔气中飞扑而来，剑修纵身跃起迎敌（Beat 3：高潮），
空中激战，剑气纵横（存在性测试：❌ 太泛，改为具体动作），
改为："剑修旋身挥剑，剑刃撕裂空气迸射红色冲击波"（具体，服务于战斗概念），

定格在剑修落地收剑，冷声道"此界之门，不容踏越"（情绪定格），

音效：剑鸣声，魔兽惨嚎，风声（服务于氛围）

禁止：水印、字幕、Logo、多余人物
```

**改进点**：
- ✅ 每个镜头都服务于 Concept（背叛→孤独→坚定）
- ✅ 删掉了"环绕快切"等炫技但不服务叙事的描述
- ✅ 台词和动作都服务于情绪转变

### 示例 2：产品广告（概念先行版）

**Step 1：定义 Concept**
```
Concept："可口可乐的夏日清凉感，从视觉（冷色调+气泡）到听觉（开瓶声+气泡声）的畅快体验"
```

**Step 2：设计 3-Beat 叙事弧**
```
Beat 1（0-2秒）：热天，角色口渴（需求铺垫）
Beat 2（3-6秒）：开瓶，倒可乐，冰块碰撞（满足需求）
Beat 3（7-10秒）：喝下可乐，满足表情+气泡特效（畅快体验）
```

**Step 3：存在性测试 + 写提示词**
```
10秒可乐广告，冷蓝白色调（服务于 Concept：清凉感），

0-2秒：热天街头，角色擦汗，看到冷饮柜中的可乐（Beat 1：需求），
镜头特写角色 thirsty 的表情（存在性测试：✅ 建立需求），

3-6秒：开瓶盖，气泡喷出，可乐倒入杯中（Beat 2：满足需求），
慢镜头展示冰块碰撞，气泡上升（存在性测试：✅ 视觉清凉感），
伴随开瓶声和气泡声（听觉清凉感），

7-10秒：角色喝下可乐，满足地呼一口气（Beat 3：畅快体验），
镜头特写可乐杯上的水珠（存在性测试：✅ 强化清凉感），
伴随满足的叹息声，

音效：开瓶声，气泡声，满足的叹息声（服务于 Concept：畅快），

禁止：文字、Logo（除非产品自带）、杂乱背景
```

**改进点**：
- ✅ 每个镜头都服务于"清凉感"概念
- ✅ 视觉（冷色调+气泡）+ 听觉（开瓶声+气泡声）双重强化概念
- ✅ 删掉了"360度旋转"等不服务概念的描述

## 概念先行 Checklist

**写提示词前，完成这个 checklist：**

- [ ] 能用一句话说清楚 Concept 吗？
- [ ] Concept 包含：角色 + 场景 + 动作 + 情绪？
- [ ] 设计了 3-Beat 叙事弧吗？
- [ ] 每个镜头都通过了存在性测试？
- [ ] 技术词（运镜/灯光/特效）都服务于 Concept？
- [ ] 删掉了不服务 Concept 的炫技描述？

**如果以上有任何一项没完成，回到 Step 1 重新定义 Concept。**

## 常见 Concept 模板

| 场景 | Concept 模板 |
|------|-------------|
| 短剧对白 | "[角色A] 与 [角色B] 的 [冲突类型]，从 [情绪A] 到 [情绪B] 的转变" |
| 产品广告 | "[产品] 的 [核心卖点]，从 [需求] 到 [满足] 的体验" |
| 仙侠战斗 | "[角色] 的 [动机]，在 [场景] 中与 [对手] 的 [冲突类型]" |
| MV/音乐卡点 | "[音乐情绪] 的视觉化，[节奏类型] 的卡点剪辑" |
| 科普教学 | "[知识点] 的可视化解释，从 [问题] 到 [答案] 的过程" |
