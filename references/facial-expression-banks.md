## 使用说明

### 表情提示词公式（进阶版）

一条好的表情提示词 = 拆成 **5 个部分**：

1. **眼神**：看向哪里，是躲闪、发亮、失焦，还是盯住对方
2. **眉毛**：是放松、上扬、压低，还是紧紧皱起
3. **嘴角和嘴唇**：是抿住、颤抖、张开，还是压住笑
4. **身体反应**：后退、前倾、攥紧手、肩膀放下
5. **剧情状态**：刚听到真相、准备告别、被背叛、终于释然

```
角色设定 + 表情提示词 + 场景情绪 + 镜头景别 + 光影风格
```

### 嵌入方法
在角色主体描述后面，加上 "表情："，或直接将公式融入提示词末尾。

### 最佳实践
1. **搭配角色描述使用** — 表情提示词需嵌入完整角色描述
2. **光影是表情的放大器** — 同一种表情 + 不同光影 = 不同效果
3. **中英文混写效果更佳** — 关键表情词建议中英文混写
4. **动态幅度控制** — 视频提示词加"动作幅度小，自然流畅"
5. **一条镜头只选一种主情绪** — 不要在一个镜头里塞三种情绪

## 0. 基础微表情（10种）

> 来源：知乎专栏。让人物不像木头人的基础表情。

### 0.1 平静克制
```
眼神稳定，眉头放松，嘴唇自然闭合，
脸上没有明显笑意，像把情绪压在心里
```

### 0.2 轻微疑惑
```
眉头轻轻皱起，眼睛微微眯起，嘴唇略微张开，
像刚听到一句不确定的话
```

### 0.3 认真倾听
```
目光集中看向对方，脸部放松，身体微微前倾，
像在捕捉每一个细节
```

### 0.4 若有所思
```
视线稍微偏向一侧，眉眼安静，嘴角收住，
像在心里快速推演答案
```

### 0.5 欲言又止
```
嘴唇微微张开又停住，眼神闪烁，喉结轻动，
像有话不敢说出口
```

### 0.6 强装镇定
```
脸上保持平静，眼底却有细微紧张，手指轻轻收紧，
像在努力不暴露情绪
```

### 0.7 礼貌微笑
```
嘴角轻轻上扬，眼神温和但不过分热情，
适合初次见面或表面客气的场景
```

### 0.8 尴尬停顿
```
笑容僵在脸上，眼神短暂躲闪，嘴角不自然地抬起，
像不知道怎么接话
```

### 0.9 不动声色
```
面部几乎没有变化，眼神却冷静观察对方，
适合隐藏真实目的的人物
```

### 0.10 暗自判断
```
眼睛快速扫过对方表情，眉头微收，嘴角压低，
像已经发现了问题
```

## 🎬 实战：分镜嵌套示例

> 来源：知乎专栏。以"雨夜院门前，男主发现师门被灭，女主隐瞒真相"为例。

把情绪拆成 4 个镜头，情绪从克制 → 震惊 → 痛苦 → 决绝，层层递进：

### 镜头 1：强装镇定（中景）
```
男主站在雨夜院门前，脸上保持平静，眼底却有细微紧张，
手指轻轻收紧，镜头中景，冷蓝雨夜光
```

### 镜头 2：发现真相（近景 → 面部特写）
```
男主看到女主手里的玉佩，表情从疑惑转为震惊，
眼睛慢慢睁大，嘴唇微张，镜头缓慢推近面部
```

### 镜头 3：含泪愤怒（面部特写）
```
男主眼里有泪但眼神很硬，嘴角发抖，
表情既痛苦又不肯退让，雨水顺着脸颊滑落
```

### 镜头 4：下定决心（中景 → 背影）
```
男主缓慢抬头，眼神变得坚定，嘴唇闭紧，眉头微收，
转身走入雨夜，背影决绝
```

## ⚠️ 新手避坑指南

> 来源：知乎专栏。新手最容易踩的 5 个坑。

### ❌ 坑 1：情绪太满
不要每条都写"极度、强烈、震撼"。情绪太满反而没有层次。

### ❌ 坑 2：一个镜头塞三种情绪
不要一条镜头里又开心又愤怒又害怕。**先确定这一镜的主情绪**。

### ❌ 坑 3：只写脸
手、肩膀、呼吸、后退半步——身体语言往往比面部更能让表演真实。

### ❌ 坑 4：忽略镜头景别
表情戏最好用**近景或面部特写**，远景很难看清情绪变化。

### ❌ 坑 5：失败就重抽
不要每次失败都重抽。先判断问题在哪：
- 角色图不行？→ 换角色图
- 表情词太粗？→ 细化表情描述
- 镜头距离不对？→ 改用近景
- 场景情绪不匹配？→ 调整光影氛围

## 💡 使用建议

### 表达式蒙版技巧
使用 Stable Diffusion 时，先用 "expression sheet"（表情表）生成同一角色的多种表情参考图，再挑选最满意的一张作为种子图（seed），用图生图模式微调。

### 动态幅度控制
从静态表情生成视频动态片段时，在表情提示词基础上加 "动作幅度小，自然流畅（subtle and natural movement）"，避免 AI 生成过于夸张的面部变形。

### 角色一致性
同一角色在不同场景中保持表情风格一致：
- 标注角色编号或名称
- 统一光影和画风描述
- 同一来源的参考图确保面部特征连贯

### 表情递进法则
短剧的"戏"来自情绪递进，而不是突然爆发：
1. 确定角色当前需要的主要情绪
2. 搭配符合角色性格的表情强度
3. 不同镜头间情绪要有层次变化
4. 用光影辅助强化情绪递进
