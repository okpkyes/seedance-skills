# 图生图变换模板 - Image Transformation Templates

本文档收录图生图（image-to-image）变换场景的完整提示词模板，解决「2D→3D」「2D→真人」等图片风格变换需求。

> 吞噬自 2d-to-3d-prompt + 2d-to-realperson-prompt 两个独立 Skill，整合为统一参考。

---

## 使用时机

**读取本文件的场景：**
- ✅ 用户要将 2D 插画 / 动漫转为 3D CGI 商业摄影风格
- ✅ 用户要将动漫 / 二次元人物转为真人写实摄影风格
- ✅ 用户提到 "2D转3D"、"立体化"、"CGI化"、"3D商业摄影"
- ✅ 用户提到 "2D转真人"、"动漫转真人"、"二次元转写实"、"真实化"
- ✅ 图生图任务中需要将平面插画升级为写实 3D 或真人质感

**注意：** 实际执行图生图时，须先调用 seedance 主流程对提示词进行优化，再使用 ImageGen 等工具执行。用户平台已支持真人照片（2026-07-09 起），默认按真人拍摄提示词写。

---

## 通用变换模板结构

所有图生图变换提示词遵循以下结构：

```
【身份保留】+【技术参数框架】+【负向约束】+【输出规范】
```

| 模块 | 作用 | 核心原则 |
|------|------|---------|
| 身份保留 | 锁定原图核心特征 | 严格复刻构图/人物/意境，1:1 情绪克隆 |
| 技术参数 | 定义目标风格的技术框架 | 光影+人物+材质+相机协同工作 |
| 负向约束 | 禁止不想要的效果 | 枚举式禁止，覆盖风格/皮肤/五官/肢体/场景 |
| 输出规范 | 定义格式与清理要求 | 画幅+HDR+去水印 |

## 模板一：2D → 3D CGI 商业摄影

**适用**：插画 / 概念图 → 3D CGI 商业摄影作品
**核心**：建立高标准技术框架，让插画中所有元素在 3D 躯壳中复活

### 完整提示词

```
将此插画转化为一幅顶级专业 CGI 商业摄影作品。所有技术参数（光影、人物、材质、构图）必须协同工作，共同服务于原插画的意境升华，绝不能允许任何单一技术的堆砌破坏画面的有机统一。核心是建立一个高标准的技术框架，让插画中的所有元素在 3D 躯壳中复活。

【画面基础：构图与意境还原】
严谨复刻： 严格保留原图的构图比例、物体摆放、人物姿态、以及核心元素的超现实概念。
深层意境： 分析原插画的情感基调（如：清幽、壮丽、深邃、通透），确保 3D 画面在"情绪感官"上保持 1:1 的完美克隆。

【光影系统：环境增强与主体塑造 (均衡处理)】
环境光适配： 3D 布光必须基于原图的逻辑。如果原图环境幽深，就用精致的局域光塑造暗部层次；如果原图环境开阔，就用高调光塑造通透感。核心是顺应，而非改变环境逻辑。
电影感塑造： 利用精准的环境光遮蔽 (AO) 塑造物体厚度。加入柔和、扩散的边缘光来分离主体，并根据原图基调引入自然的体积光效应（如水汽、风沙、星尘），以增强空间深度。
负向约束： 严禁引入任何硬朗、过曝、或违背原插画深层意境的通用补光或程式化高光。

【人物与主体：现实化与微细节】
形体优化： 自动修正形体和解剖结构（例如：手指、关节比例、畸变），确保完全符合现实生理逻辑。
写实皮肤： 人物皮肤（如有）必须呈现通透、自然、微孔清晰的毛孔纹理，具备真实的次表面散射 (SSS) 质感，避免塑料感。
毛发与眼神： 发丝根根分明，具有真实光泽；瞳孔需有深度和真实的虹膜纹理。

【材质与细节：物理升华】
写实质感： 所有材质（织物、金属、植物、道具）需根据原图色相基调，赋予对应的基于物理的渲染 (PBR) 属性（反射、粗糙度、金属度、折射率）。
细节放大： 放大查看时，重工刺绣的丝线纹路、金属器具的微小划痕、道具的复杂结构必须清晰可见。

【相机设定：商业摄影级】
焦距： 85mm 黄金摄影焦距。
景深： f/1.8 大光圈效果，营造极浅景深，确保视觉中心极度锐利，背景呈现符合物理光学规律的柔和焦外光斑 (Bokeh)。

【输出规范】
9:16，高动态范围 (HDR)，去原图水印、文字、杂质。
```

### 技术模块拆解

| 模块 | 核心指令 | 关键技术点 |
|------|---------|-----------|
| 画面基础 | 构图与意境还原 | 比例复刻 + 情绪 1:1 克隆 |
| 光影系统 | 环境增强与主体塑造 | AO、边缘光、体积光、负向约束 |
| 人物主体 | 现实化与微细节 | 解剖修正、SSS 皮肤、毛发虹膜 |
| 材质细节 | 物理升华 | PBR 属性（反射/粗糙度/金属度/折射率）、微距细节 |
| 相机设定 | 商业摄影级 | 85mm、f/1.8、浅景深 Bokeh |
| 输出规范 | 格式与清理 | 9:16、HDR、去水印 |

### 可调参数

| 参数 | 默认值 | 候选值 |
|------|--------|--------|
| 画幅比例 | 9:16 | 1:1、16:9、4:3、3:4 |
| 焦距 | 85mm | 50mm（更广）、135mm（更压缩） |
| 光圈 | f/1.8 | f/1.4（更浅）、f/2.8（更深） |
| 情感基调 | 按原图分析 | 清幽 / 壮丽 / 深邃 / 通透 / 温暖 |
| 体积光效应 | 按原图适配 | 水汽 / 风沙 / 星尘 / 光束 / 无 |

## 模板二：2D 动漫 → 真人写实摄影

**适用**：动漫 / 二次元人物 → 真实人像摄影
**核心**：精准保留人物身份特征，赋予真实骨骼结构、皮肤质感、毛发细节

### 完整提示词

```
必须精准保留原动漫图片中的人物身份特征、脸部轮廓、五官比例、样貌气质、发型的写实摄影质感。

人物要求:
将动漫人物转化为真实存在的人类形象，脸部必须具有真实的人类骨骼结构、自然面部轮廓、真实但好看的五官比例、真实皮肤质感、自然肤色、细腻的皮肤纹理、真实眼睛反光、真实睫毛和眉毛细节。保留原图人物的脸型特征、五官气质和整体样貌印象，但不要保留任何二次元夸张比例，不要动漫大眼，不要塑料感，不要AI假脸。

皮肤要求:
皮肤必须呈现真实摄影级别质感，但整体状态要偏细腻、嫩、干净、自然、有轻微通透感，不要粗糙，不要颗粒过重，不要毛孔夸张，不要老化感，不要蜡像感，也不要过度磨皮。皮肤纹理要清晰但克制，属于高质量年轻肌肤的真实质感；细腻、平滑、柔和、富有弹性，保留轻微真实纹理与自然明暗起伏。除非原角色本身设定明显年龄较大，否则不要生成成熟粗糙、松弛、厚重或沧桑的皮肤状态。

发型要求:
发型必须完全参考原动漫图的长度、刘海、发丝走向、层次、卷度、蓬松感和颜色，将其转化为现实中的真实头发质感。头发要有真实发丝细节、自然光泽、柔软蓬松感和真实物理垂坠感，不要像假发，不要僵硬，不要动漫块状头发。

服装要求:
服装必须严格参考原动漫图的款式、剪裁、颜色、图案、材质、装饰细节和穿着方式风格。

画面风格:
极致写实摄影风格，真实世界照片质感，电影级真实光影，高动态范围，真实镜头景深，真实空气感，真实环境反射，细节清晰，高分辨率，8K超清，专业摄影构图。整体效果必须像现实世界中真实拍摄的人像照片，而不是动漫、插画、游戏CG或3D角色渲染图。

构图要求:
保持原图的画面构图、人物大小、镜头角度、景别、透视关系和主体位置。高清真实摄影效果。

禁止内容:
禁止动漫风、插画风、二次元风、3D卡通风、游戏CG风、塑料皮肤、假人感、过度磨皮、皮肤粗糙过度、毛孔夸张、老化感过强、眼睛颜色错误、虹膜颜色偏差、五官变形、眼睛过大、脸部模糊、手指畸形、肢体扭曲、服装细节丢失、场景随意更换、背景虚假、光线不统一、明显AI感、低清晰度、过度美颜、过度锐化。

最终效果:
最终效果必须像一张真实摄影师在现实世界中拍摄的高质量照片。人物要真实、自然、好看，皮肤细腻嫩感明显，但依然保留真实感和高级质感。
```

## 两套模板的区别

| 维度 | 2D→3D CGI | 2D→真人 |
|------|-----------|---------|
| 目标 | CGI 商业摄影 | 真实人像摄影 |
| 质感 | PBR 材质、3D 渲染 | 真实皮肤、真实骨骼 |
| 光影 | AO + 体积光 + 边缘光 | 电影级真实光影 |
| 相机 | 85mm f/1.8 Bokeh | 真实镜头景深 |
| 皮肤 | SSS、PBR | 6维平衡（细腻/嫩/干净/通透） |
| 适用 | 插画/概念图 → 3D 商业片 | 动漫/二次元 → 写真人像 |

## 使用流程

1. **分析原图：** 识别构图比例、核心元素、情感基调、人物身份特征
2. **选择模板：** 根据变换目标选择「2D→3D」或「2D→真人」模板
3. **应用模板：** 使用对应完整提示词，根据原图特征调整可调参数
4. **seedance 优化：** 通过 seedance 主流程对提示词进行最终优化（强制）
5. **执行生成：** 使用 ImageGen 等工具执行图生图
6. **附录输出：** 在回复中附上优化后的完整提示词供用户复用
