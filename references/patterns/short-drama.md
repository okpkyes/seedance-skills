# 短剧/对白场景模式库

短剧、对话场景的现成提示词模板，直接套用或微调。

---

## 模式 1：霸总短剧名场面

**适用场景**：女主撕契约、男主悔悟、打脸反转等经典霸总剧情节

**提示词模板**：
```
画面（0-5秒）：[具体动作描述，如"特写女主撕契约镜头"]，[环境描述]，
[角色A动作]，

台词1（角色A，[情绪]）："[对话内容]"

画面（6-10秒）：[镜头变化，如"镜头切换到角色B"]，
[角色B反应动作]，

台词2（角色B，[情绪]）："[对话内容]"

画面（11-15秒）：[结尾画面，如"角色B僵住，女主转身离开"]，

音效：[背景音乐风格] + [关键音效，如"契约撕碎声"]

时长：精准15秒
```

**参考素材**：
- @图片1（可选）：[角色形象参考图]
- @图片2（可选）：[场景参考图]

**参数建议**：
- 时长：15秒（标准短剧时长）
- 比例：9:16（竖屏，适合抖音/快手）
- 风格：国产短剧、霸总剧

**示例 - 女主撕契约反杀**：
```
画面（0-5秒）：特写女主撕契约镜头，纸屑飘落，总裁单膝跪地伸手阻拦，眼神慌乱，
女主侧身躲开，嘴角挂着冷漠笑意

台词1（总裁，卑微慌乱）："苏晚！契约还没结束，你不能走！我给你钱，给你地位！"

画面（6-10秒）：女主抬脚避开他的手，将撕碎的契约纸扔在他脸上，
镜头扫过周围宾客的窃窃私语

台词2（女主，冷漠反杀）："契约？顾总，当初是你说，我连给你提鞋都不配，现在求我？晚了！"

画面（11-15秒）：总裁僵在原地，脸上沾着纸屑，女主转身昂首离开，红裙裙摆飘动

音效：华丽又带张力的背景音，契约撕碎的声响，宾客轻微的窃窃私语声

时长：精准15秒
```

## 模式 2：古风仙侠对白

**适用场景**：仙侠剧、古装剧的对话场景

**提示词模板**：
```
[时代背景]风格，[场景描述，如"竹林/宫殿/山顶"]，

画面（0-X秒）：[镜头语言，如"全景推到中景"]，
[角色A][动作描述]，[表情描述]，

台词1（角色A，[情绪]）："[古风对话，文雅或霸气]"

画面（X-Y秒）：[镜头变化]，
[角色B][动作反应]，

台词2（角色B，[情绪]）："[对话内容]"

画面（Y-结束）：[结尾画面，如"两人对视/转身离开/拔剑"]，

音效：[古风背景音乐] + [环境音，如"竹林风声/剑鸣/水流"] 

色调：[如"青绿古画色调/金红暖色调"]
```

**参考素材**：
- @图片1：[角色A古装造型参考]
- @图片2：[角色B古装造型参考]
- @图片3：[场景参考，如竹林/宫殿]

**参数建议**：
- 时长：10-15秒
- 比例：16:9（横屏，电影感）或 9:16（竖屏）
- 风格：中国奇幻动画电影风格 / 古装剧写实风格

**示例 - 师徒决裂**：
```
古风仙侠风格，山顶云海场景，

画面（0-5秒）：全景推到中景，白袍师父背对镜头站在山崖边，
衣袍被风吹动，远处云海翻涌，

台词1（师父，威严中带失望）："徒儿，你竟偷学禁术，堕入魔道。"

画面（6-10秒）：低角度仰拍，黑衣弟子缓缓转身，手中魔剑泛着暗红光芒，
眼神冰冷，嘴角冷笑，

台词2（弟子，冷漠决绝）："师父，你说正道，却容不下天下苍生。今日，我便以魔道之力，重写这天道！"

画面（11-15秒）：环绕快切，弟子纵身跃向云海，师父伸手阻拦却只抓住残影，
剑气纵横，魔气冲天，

音效：史诗感古风配乐，剑鸣声，风声，衣袍猎猎声

色调：金红暖色调（法术光效）对比冷蓝（云海）
```

## 模式 4：悬疑/犯罪对质

**适用场景**：悬疑剧、警匪剧的高潮对质场景

**提示词模板**：
```
[压抑的场景，如"昏暗审讯室/雨夜天台/废弃仓库"]，

画面（0-X秒）：[低角度/特写，营造紧张感]，
[角色A][动作，如"步步逼近/握紧拳头"]，

台词1（角色A，[情绪，如"压抑愤怒"]）："[对话，短促有力]"

画面（X-Y秒）：[快速剪辑/手持晃动镜头，增加紧张感]，
[角色B][反应，如"后退/冷汗/眼神闪烁"]，

台词2（角色B，[情绪，如"慌乱狡辩"]）："[对话]"

画面（Y-结束）：[定格/慢动作]，[高潮画面，如"拳头挥出/枪口特写"]，
```

**参数建议**：
- 时长：10-15秒
- 比例：16:9 或 9:16
- 风格：悬疑剧、电影感、高对比灯光

## 短剧提示词技巧

### 台词写作规则

1. **用引号包裹台词**：`"对话内容"`
2. **标注角色和情绪**：`台词（角色名，情绪）："对话"`
3. **台词要短而有张力**：单句不超过20字，适合15秒视频
4. **情绪要夸张**：短剧需要夸张的情绪表达（撕心裂肺/冷漠狠厉/卑微慌乱）

### 分镜节奏控制

| 时长 | 分镜数 | 节奏 |
|------|--------|------|
| 4-8秒 | 1-2个镜头 | 慢，特写为主 |
| 9-12秒 | 2-3个镜头 | 中速，中景+特写 |
| 13-15秒 | 3-4个镜头 | 快，快切+镜头运动 |

### 情绪递进结构

```
开头（0-3秒）：铺垫情绪，建立冲突
中段（4-10秒）：情绪升级，对话交锋
高潮（11-13秒）：转折点，金句/反转
结尾（14-15秒）：定格情绪，留悬念
```

### 音效设计

**必备音效**：
- 背景音乐：匹配情绪（紧张/抒情/史诗/华丽）
- 关键动作音：撕纸声/摔门声/脚步声
- 环境音：雨声/风声/人群嘈杂声

**音效描述格式**：
```
音效：背景音乐风格 + 关键动作音 + 环境音
```
