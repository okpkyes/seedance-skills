# 电商广告模式库

电商/产品广告的现成提示词模板，直接套用或微调。

---

## 模式 1：产品 360 度展示

**适用场景**：产品主图视频、电商详情页视频

**提示词模板**：
```
@图片1中的[产品名]，360度高速旋转2圈后，突然停住蓄力分裂成了3个部分进行展示。
随后分解后的[产品名]的上中下三部分快速向内旋转合成，一罐完整的[产品名]，
3D渲染产品展示特效，动感产品特效展示
```

**参考素材**：
- @图片1：[产品正面高清图，白底或透明底]

**参数建议**：
- 时长：5-8秒
- 比例：1:1（方形）或 16:9（横屏）
- 风格：3D渲染、商业摄影

**变体**：
- 慢速旋转（奢华感）："缓慢优雅地旋转360度"
- 爆炸展示（科技感）："爆炸分解展示内部结构"
- 悬浮展示（未来感）："产品悬浮空中，光影流转"

## 模式 2：第一人称沉浸式体验

**适用场景**：美妆、食品、手冲咖啡等"使用过程"产品

**提示词模板**：
```
第一人称视角，手持[产品]，[使用动作详细描述]，
镜头贴近产品，展示质感和细节，
自然光从窗户洒入，温暖的生活氛围，
伴随真实的使用音效（倒水声/涂抹声/拆包装声）
```

**参考素材**：
- @图片1：[产品使用场景参考图]
- @音频1（可选）：[真实使用音效参考]

**参数建议**：
- 时长：8-10秒
- 比例：9:16（竖屏，适合抖音/小红书）
- 风格：生活化、Vlog风格

**示例 - 手冲咖啡**：
```
第一人称视角，手持咖啡滤杯，热水缓慢倒入，咖啡液滴落，
镜头贴近咖啡液面，展示油脂和气泡，
自然光从窗户洒入，温暖的生活氛围，
伴随滤杯滴水声和水流声
```

## 模式 3：模仿参考广告

**适用场景**：已有竞品广告或理想广告风格，想替换成自己的产品

**提示词模板**：
```
参考@视频1的[广告创意/运镜/节奏/转场效果]，
将视频中的[原产品]替换为@图片1的[你的产品]，
保持原有的[灯光/色调/节奏/情绪]不变，
产品展示时加入[特写/旋转/3D渲染]效果
```

**参考素材**：
- @视频1：[参考广告视频，包含理想的创意/运镜/节奏]
- @图片1：[你的产品高清图]

**参数建议**：
- 时长：与参考视频相同
- 比例：与参考视频相同
- 风格：与参考视频一致

**注意**：
- Seedance 2.0 会智能替换产品，但保持原有视频的"感觉"
- 如果参考视频有真人脸部，会被拦截（平台限制）

## 模式 4：产品变身/变装

**适用场景**：新品发布、产品升级、包装换新

**提示词模板**：
```
使用首尾帧控制，
@图片1作为首帧（旧产品/旧包装），
@图片2作为尾帧（新产品/新包装），
中间过渡效果：[粒子消散重聚/光影流转变换/魔法阵旋转],
流畅的产品变身过程，电影级特效
```

**参考素材**：
- @图片1：[旧产品/旧包装高清图]
- @图片2：[新产品/新包装高清图]

**参数建议**：
- 时长：4-6秒（变身过程要快）
- 比例：9:16 或 16:9
- 风格：魔法/科幻/未来感

**变体**：
- 魔法变身："魔法阵旋转，光影流转，产品缓缓变身"
- 科技变身："数字粒子消散，重组为新品"
- 简约变身："淡入淡出过渡，干净利落"

## 模式 5：美食/饮料广告

**适用场景**：餐厅、食品、饮料品牌

**提示词模板**：
```
[美食/饮料]特写镜头，[描述质感：汁水/泡沫/蒸汽/冰晶]，
慢镜头展示[倾倒/咬下/切开]瞬间，
伴随满足的音效（咀嚼声/倒水声/开瓶声），
暖色调灯光，美食摄影风格，
让人看了流口水的视觉效果
```

**参考素材**：
- @图片1：[美食/饮料高清参考图]
- @音频1（可选）：[咀嚼声/倒水声参考]

**参数建议**：
- 时长：6-10秒
- 比例：9:16（竖屏）或 1:1（方形）
- 风格：美食摄影、商业广告

**示例 - 可乐广告**：
```
可乐饮料特写，冰块碰撞，气泡上升，
慢镜头展示可乐倒入杯中的瞬间，泡沫溢出杯口，
伴随开瓶声和倒水声，
冷色调蓝光，夏日清爽感，
让人看了想喝一口的视觉效果
```

## 模式 6：奢侈品/时尚单品

**适用场景**：包包、鞋子、手表、珠宝等奢侈品

**提示词模板**：
```
[产品]特写，极简构图，大量留白，
单色背景（纯黑/纯白/米色），
侧逆光打亮产品轮廓，质感突出，
缓慢的推镜头展示细节（缝线/纹理/反光），
无音乐，只有环境音（布料摩擦声/金属碰撞声）
```

**参考素材**：
- @图片1：[产品高清图，展示质感]

**参数建议**：
- 时长：8-12秒
- 比例：9:16（竖屏）或 4:5（Instagram）
- 风格：极简、高级感、奢侈品牌广告

**关键词**：
```
极简构图、留白、侧逆光、推镜头、慢动作、
质感突出、高级感、奢侈品牌风格
```

## 通用电商提示词技巧

### 产品展示黄金法则

1. **主图视频（3-5秒）**：
   - 360度旋转 或 缓慢推进特写
   - 展示产品全貌 + 关键细节
   - 适合：淘宝主图、亚马逊主图

2. **详情页视频（10-15秒）**：
   - 分镜展示：全貌 → 细节 → 使用 → 效果
   - 时间戳分镜法控制节奏
   - 适合：电商详情页、独立站产品页

3. **社交媒体广告（6-10秒）**：
   - 前3秒抓眼球（特效/变身/惊喜）
   - 中间展示产品
   - 结尾Call to Action
   - 适合：抖音、小红书、Instagram

### 禁止项声明（电商专用）

```
禁止：
- 任何文字、字幕、Logo、水印（除非是产品自带的品牌Logo）
- 产品变形、扭曲、模糊
- 背景杂乱，干扰产品展示
- 光线过暗或过曝，看不清产品细节
```
