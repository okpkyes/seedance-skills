# 角色时装联名企划案（Character Fashion Lookbook）

一份把「动漫角色 cosplay」升级为「高级时装联名主视觉企划」的**生图提示词模板**。

> ⚠️ 性质说明：这是**图像生成**模板（竖版 9:16 写真/杂志封面），与本文档主体（Seedance 视频）互补。
> 生图时同样走 seedance 提示词工程流程，本模板提供可直接替换占位符的成品结构。

## 核心思路

不是「还原角色长相」，而是假设这个角色存在于现实世界、成为某时尚品牌代言人，用高级时装语言重新诠释她：

- 真人高定角色写真（写实 adult cosplay 模特，非 2D 还原）
- 服装设计线稿 + 配饰草图 + 面料样品 + 色卡环绕
- 材质与色板分析
- 杂志 Lookbook 排版 / 奢侈品牌宣传美学

效果从「普通 cosplay」升级为「一份角色时装联名企划案」。

## 占位符拆解

| 占位符 | 含义 | 填写要点 |
|---|---|---|
| `{{角色名}}` | 目标角色 | 只换名字，其余结构不动 |
| `{{作品名}}` | 出处 IP | 如《海贼王》《原神》 |
| `服装重新设计` | 标志性元素提炼 | 把角色视觉符号列成「、」清单，并加「高级时装化」改写（如「度假风短上装」「海洋珠宝元素」） |
| `其他详细信息` | 质感锚点 | 固定即可，不必改 |

## 使用方式

1. 把上方通用结构整段发给 AI（或粘贴本模板）。
2. 告诉它「把 `{{角色名}}` 换成 XX，《作品名》换成 YY」。
3. AI 会生成该角色的专用版 Prompt（针对单一角色深度优化，还原度/稳定性优于通用版）。
4. 投到即梦/可灵/Seedance 等支持真人写实的平台生图。

## 适配技巧（平台过审 / 提升质感）

- **真人写实口径**：保留「逼真的东亚成年女性 cosplay 模特」+「自然的肌肤质感」，避免平台把角色判为未成年。
- **占比控制**：「模特占据画面 92%-96%」确保全身入镜、不被草图遮挡主体。
- **环绕草图数量**：6-7 幅最佳；少于 5 会偏普通写真，多于 8 容易杂乱。
- **换风格**：把「韩式时尚杂志风格」替换为「法式优雅 / 美式街头 / 极简高定」即可切换品牌调性。
- **视频延展**：若要做成动态 Lookbook，可拆成「定帧写真 → I2V 微动（发丝飘动/衣角轻摆）」，走 SVI 无限视频链路。
