# 电影感语汇库

为 Seedance 2.0 提示词提供专业影视词汇，提升生成质量。

## 使用原则

**每次生成提示词时，从下列词汇库挑选 5-8 个高信号 token 嵌入提示词。**

选择依据：
- 任务类型（电影感/广告/短视频/MV）
- 情绪氛围（紧张/浪漫/史诗/温馨）
- 视觉风格（写实/奇幻/赛博朋克/古风）

**注意**：Seedance 2.0 对中文自然语言理解强，但添加专业词汇可显著提升质量。

## 镜头语言词汇（扩展版）

### 景别

| 词汇 | 说明 | 使用场景 |
|------|------|---------|
| 大远景 (Extreme Long Shot) | 环境为主，人物很小 | 开场、结尾、环境展示 |
| 远景 (Long Shot) | 人物全身+环境 | 动作场面、追逐戏 |
| 全景 (Full Shot) | 人物全身+少量环境 | 对话、站立 |
| 中景 (Medium Shot) | 腰部以上 | 对话、访谈 |
| 近景 (Close-Up) | 胸部以上 | 情绪表达 |
| 特写 (Close-Up) | 脸部占据大部分画面 | 微表情、情绪高潮 |
| 大特写 (Extreme Close-Up) | 眼睛、嘴巴等局部 | 紧张、悬疑 |
| 超特写 (Macro) | 极细微细节 | 产品细节、纹理 |

| 移镜头 (Truck/Track) | 镜头平行移动 | ✅ 强 |
| 跟拍 (Follow Shot) | 跟随主体移动 | ✅ 强 |
| 手持跟拍 (Handheld) | 手持摄像机跟随 | ✅ 强 |
| 环绕拍摄 (Circular Shot/Orbit) | 绕主体360度 | ✅ 强 |
| 航拍 (Aerial/Bird's Eye) | 俯拍全景 | ✅ 强 |
| 低角度 (Low Angle) | 从下往上拍 | ✅ 强 |
| 仰拍 (Tilt Up) | 镜头向上倾斜 | ✅ 强 |
| 俯拍 (Tilt Down) | 镜头向下倾斜 | ✅ 强 |

## 使用建议

### 提示词结构模板

```
[时长] [画幅比] [帧率] [色调总纲]

[时间戳分镜描述]

镜头语言: [景别] + [运镜] + [角度]
灯光: [灯光类型] + [光比]
色彩: [色调风格] + [色彩分级]
效果: [VFX/大气效果]

台词: [角色名(情绪)]："对话内容"
音效: [音效描述]

禁止: [不需要的元素]
```

### 示例：仙侠战斗提示词（应用语汇库）

```
15秒仙侠高燃战斗，金红暖色调，电影感

0-3秒：低角度大特写，主角蓝袍衣摆被热浪吹动，双手紧握雷纹巨剑，
剑刃赤红电光持续爆闪（电弧效果），地面熔岩翻涌（大气效果：热浪扭曲），
参考王家卫的饱和色彩，侧逆光打亮主角轮廓，
主角低喝"今日，便以这柄剑，镇尔等邪祟！"（台词标注情绪：坚定）

4-8秒：环绕拍摄快切（运镜：环绕），主角旋身挥剑（动作描述），
剑刃撕裂空气迸射红色冲击波（能量波效果），
前排魔兵被击飞碎裂成灰烬粒子（粒子消散效果），
镜头浅景深突出主角，背景虚化，
伴随剑气破空声与魔兵惨嚎（音效描述）

9-12秒：仰拍拉远定格慢放（运镜：仰拍+慢动作），
主角跃起腾空，剑刃凝聚巨型雷光电弧劈向魔兵群（电弧+能量波），
体积光穿透云雾（大气效果：体积光），

13-15秒：缓推特写主角落地收剑的姿态（运镜：推镜头+慢动作），
衣摆余波微动，冷声道"此界之门，不容踏越"（台词情绪：冷酷），
音效收束为余音震颤与渐弱风声（音效：渐弱），

禁止：文字、水印、Logo、多余人物、现代元素
```

**改进点**：
- ✅ 添加了导演风格参考（王家卫）
- ✅ 明确了镜头语言（低角度、环绕、仰拍、推镜头）
- ✅ 添加了灯光描述（侧逆光）
- ✅ 添加了VFX效果（电弧、能量波、粒子消散、体积光）
- ✅ 添加了摄影技术（浅景深、定格慢放）
- ✅ 音效描述更专业（余音震颤、渐弱）
