# 模特一致性 8 变体（同一张脸·8 种动作/表情）

锁定**同一张脸 + 同一发型轮廓 + 同一肤色肤质**，8 种变体动作/表情。最稳的角色一致性锚点。

## 完整提示词

```
8 格分镜，同一位模特在 8 种不同动作与表情下的变体：

1. 正面轻笑，微微露齿，眼神自信。
2. 看向镜头，嘴唇轻抿，神情明艳但克制。
3. 一只手轻碰嘴唇，带一点时装俏皮感。
4. 头部轻微侧转，眼神看向镜头外，像拍杂志抓拍。
5. 手掌轻贴脸颊，笑容自然，面部肌肉真实。
6. 头轻轻歪向一侧，眼神柔和，带一点甜酷感。
7. 轻微张嘴，目光松弛，像高定广告定格。
8. 抬手整理头发或挡光，姿态放松，神情高级。

重点：保持同一位模特，同一张脸，同一眼型与唇形，同一发型轮廓，同一肤色和肤质，不跑脸，不换脸，不幼态化。
```

## 8 变体速查

| # | 动作 | 表情关键词 | 情绪标签 |
|---|---|---|---|
| 1 | 正面轻笑 | 微微露齿、眼神自信 | 自信 |
| 2 | 看向镜头 | 嘴唇轻抿、明艳克制 | 高冷 |
| 3 | 手碰嘴唇 | 时装俏皮感 | 俏皮 |
| 4 | 头部侧转 | 看向镜头外、杂志抓拍 | 自然 |
| 5 | 手贴脸颊 | 笑容自然、肌肉真实 | 温柔 |
| 6 | 头歪一侧 | 眼神柔和、甜酷感 | 甜酷 |
| 7 | 轻微张嘴 | 目光松弛、广告定格 | 高级 |
| 8 | 抬手整理头发/挡光 | 姿态放松、神情高级 | 高级感 |

## 一致性铁律

- 同一位模特（同一张脸）
- 同一眼型 + 唇形
- 同一发型轮廓（silhouette，不只是颜色）
- 同一肤色 + 肤质
- ❌ 不跑脸 / 不换脸 / 不幼态化

> "不幼态化"是显式合规约束：避免平台把"成熟模特"渲染成幼态脸。

## 使用方式

1. 先用首张图锁定参考脸（`seedance` 的 `@引用` 或 Midjourney 的 `--cref`/SD 的 IP-Adapter）。
2. 把参考图 + 上述 8 变体提示词投到生图模型。
3. 适用于：
   - 角色资料卡的 8 个标准变体
   - 杂志/广告的"同一主角多角度"主视觉
   - 短剧/漫剧同一角色不同场次的形象锚点
   - 模特档口/电商商品页的同款多图
