# 面部三视图（影楼测试三联图·灰幕布 A/B/C 版）

把任意人像换成**专业影楼测试三联图**的标准结构：正面 / 3/4 侧面 / 90° 侧面，A/B/C 标记。

## 完整提示词

```
无论参考图像中出现何种背景（白色、户外或任何环境）都将其替换，

最终渲染为高级影楼人像测试三联图，背景为带纹理的灰色无缝幕布。

灰色背景需带有柔和渐变衰减与细腻的摄影颗粒。

雪金面板中保持完全的花博元对生成张会幅并排的中特写镜头的正面视角-C 面板（右侧）45° 四分之三侧面视角标准 90° 纯侧面视角。

用两条细竖线分隔面板，线条笔直、间距均匀，呈现干净的版式设计。

统一为受控影棚布光：柔和但有方向性的主光、自然的阴影塑造、真实的明暗对比、清晰的眼部细节、写实的皮肤质感（禁止过度磨皮）。

人物的身份、面部比例、发型与整体造型需与参考图完全匹配，

但所有面板的背景必须统一为同款带纹理的灰色影棚幕布。

仅需添加面板标注：

在左面板上方标注 "A"，中间面板上方标注 "B"，右面板上方标注 "C"，

字体为简洁干净的无衬线字体，字号小巧、风格低调。

禁止添加任何其他文字、水印、Logo、字幕、UI 元素、边框等额外内容。
```

## 拆解要点

| 维度 | 固定写法 |
|---|---|
| 背景 | 灰色无缝幕布（带柔和渐变 + 摄影颗粒） |
| 三联 | A=正面 / B=3/4 右侧 45° / C=90° 纯侧 |
| 分隔 | 两条细竖线，笔直等距 |
| 布光 | 受控影棚：柔和主光 + 自然阴影 + 明暗对比 |
| 人物 | 身份、比例、发型、造型与参考图完全匹配 |
| 标注 | 仅 A/B/C 三字母，无衬线、小字号、低调 |
| 禁用 | 任何其他文字、水印、Logo、字幕、UI、边框 |

## 使用方式

1. 把"参考图"+ 上述提示词投到生图模型。
2. 适用于：模特/角色资料卡定档、模型建档、形象参考集、合同交付定妆照。
3. 灰幕布版适合做"光影/造型对比"——同一张脸三种角度，方便后续做角色一致性锚定。

## 触发词

- 三视图、影楼测试图、影棚三联、A/B/C面板、character turnaround、model sheet、test shot
